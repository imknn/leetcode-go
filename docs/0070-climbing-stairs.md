## 题目描述

假设你正在爬楼梯。需要 `n` 阶你才能到达楼顶。

每次你可以爬 `1` 或 `2` 个台阶。你有多少种不同的方法可以爬到楼顶呢？

**示例 1：**
```
输入：n = 2
输出：2
解释：有两种方法可以爬到楼顶。
1. 1 阶 + 1 阶
2. 2 阶
```

**示例 2：**
```
输入：n = 3
输出：3
解释：有三种方法可以爬到楼顶。
1. 1 阶 + 1 阶 + 1 阶
2. 1 阶 + 2 阶
3. 2 阶 + 1 阶
```

**提示：**
- `1 <= n <= 45`

## 解题算法

### 方法：动态规划（滚动变量）

**核心思路：**
爬楼梯问题本质上是斐波那契数列。到达第 n 阶的方法数等于到达第 n-1 阶和第 n-2 阶方法数之和：
```
f(n) = f(n-1) + f(n-2)
```
其中 `f(1) = 1`，`f(2) = 2`。

使用两个变量 `a`、`b` 滚动记录前两个状态，避免使用数组。

**代码实现：**
```go
func climbStairs(n int) int {
	if n <= 2 {
		return n
	} else {
		a, b := 1, 2
		for ; n > 2; n-- {
			a, b = b, a+b
		}
		return b
	}
}
```

**复杂度分析：**
- 时间复杂度：O(n)，循环 n-2 次
- 空间复杂度：O(1)，只使用两个变量
