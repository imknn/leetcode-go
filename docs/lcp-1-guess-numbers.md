## 题目描述

小A 和 小B 在玩猜数字。小B 每次从 1, 2, 3 中随机选择一个，小A 每次也从 1, 2, 3 中随机选择一个猜测。他们一共猜了 `guess` 次，每次猜完后小B 会告诉小A 他这次猜的数是否正确（即是否猜中了小B 当次选的数）。

请返回小A 猜对了多少次。

**示例 1：**

```
输入：guess = [1,2,3], answer = [1,2,3]
输出：3
解释：小A 三次都猜对了。
```

**示例 2：**

```
输入：guess = [2,2,3], answer = [3,2,1]
输出：1
解释：小A 只有第二次猜对了。
```

## 解题算法

### 方法：逐位比较

**核心思路：**
依次比较两个数组对应位置的元素，统计相同位置上值相等的个数。由于题目限定只猜 3 次，直接遍历 3 个位置即可。

**代码实现：**
```go
func game(guess []int, answer []int) int {
	num := 0
	for i := 0; i < 3; i++ {
		if guess[i] == answer[i] {
			num++
		}
	}
	return num
}
```

**复杂度分析：**
- 时间复杂度：O(1)，固定遍历 3 个元素
- 空间复杂度：O(1)，只使用了常数级别的额外空间
