## 题目描述

给你一个整数 `n`，请你判断该整数是否是 2 的幂次方。如果是，返回 `true`；否则，返回 `false`。

一个整数是 2 的幂次方当且仅当可以写成 `2^x` 的形式，其中 `x` 是整数。

**示例 1：**
```
输入：n = 1
输出：true
解释：2^0 = 1
```

**示例 2：**
```
输入：n = 16
输出：true
解释：2^4 = 16
```

**示例 3：**
```
输入：n = 3
输出：false
```

**提示：**
- `-2^31 <= n <= 2^31 - 1`

## 解题算法

### 方法：位运算

**核心思路：**
2 的幂在二进制表示中只有一个 1。利用 `n & (n-1)` 可以消去最低位的 1，如果结果为 0 且 `n > 0`，则说明 `n` 是 2 的幂。

例如：
- `n = 8 (1000)`, `n-1 = 7 (0111)`, `n & (n-1) = 0` -> 是 2 的幂
- `n = 6 (0110)`, `n-1 = 5 (0101)`, `n & (n-1) = 0100 != 0` -> 不是 2 的幂

**代码实现：**
```go
func isPowerOfTwo(n int) bool {
	// 100 011
	return n > 0 && (n&(n-1) == 0)
}
```

**复杂度分析：**
- 时间复杂度：O(1)，只进行常数次位运算
- 空间复杂度：O(1)，只使用常量额外空间
