## 题目描述

给定一个大小为 `n` 的数组 `nums`，返回其中的多数元素。多数元素是指在数组中出现次数 **大于** `n/2` 的元素。

你可以假设数组是非空的，且给定的数组总是存在多数元素。

**示例 1：**

```
输入：nums = [3,2,3]
输出：3
```

**示例 2：**

```
输入：nums = [2,2,1,1,1,2,2]
输出：2
```

## 解题算法

### 方法：哈希表计数

**核心思路：**
使用哈希表统计每个数字出现的次数。在遍历过程中，一旦发现某个数字的出现次数超过了 `n/2`，立即返回该数字。由于多数元素的出现次数一定大于其他所有元素出现次数之和，因此可以在遍历过程中提前返回结果。

**代码实现：**
```go
func majorityElement(nums []int) int {
	m := make(map[int]int)
	s := len(nums) / 2
	for _, n := range nums {
		if _, ok := m[n]; ok {
			m[n]++
		} else {
			m[n] = 1
		}
		if m[n] > s {
			return n
		}
	}
	return nums[0]
}
```

**复杂度分析：**
- 时间复杂度：O(n)，遍历数组一次
- 空间复杂度：O(n)，哈希表最多存储 n 个不同的元素
