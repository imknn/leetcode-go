## 题目描述

给定一个非负整数 num，反复将各个位上的数字相加，直到结果为一位数。返回这个结果。

**示例 1：**
```
输入：num = 38
输出：2
解释：各位数字相加的过程为：3 + 8 = 11，1 + 1 = 2。由于 2 是一位数，所以返回 2。
```

**示例 2：**
```
输入：num = 0
输出：0
```

## 解题算法

### 方法：模拟逐位相加

**核心思路：**
循环将整数的每一位数字提取出来相加，直到结果为一位数（小于 10）。每次循环中，通过取模 10 和整除 10 逐位提取数字并累加。

注：此题还存在数学公式解法 `(num-1)%9 + 1`（数字根），代码中也有注释。

**代码实现：**
```go
func addDigits(num int) int {
	m, count := num, 0
	for m >= 10 {
		count++
		m = 0
		for num > 0 {
			m += num % 10
			num /= 10
		}
		num = m
	}
	return m
}
```

**复杂度分析：**
- 时间复杂度：O(log(num))，每次循环使数字位数减少
- 空间复杂度：O(1)，只使用了常数个额外变量
