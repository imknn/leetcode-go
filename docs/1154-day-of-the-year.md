## 题目描述

给你一个字符串 date，按 "YYYY-MM-DD" 格式表示一个现行公元纪年法日期。请你计算并返回该日期是当年中的第几天。

**示例 1：**
```
输入：date = "2019-01-09"
输出：9
解释：给定日期是 2019 年的第 9 天。
```

**示例 2：**
```
输入：date = "2019-02-10"
输出：41
```

**示例 3：**
```
输入：date = "2003-03-01"
输出：60
```

## 解题算法

### 方法：前缀和 + 闰年判断

**核心思路：**
预先计算每个月之前所有月份的累计天数（前缀和），然后加上当前月份的日期即可得到该天是第几天。闰年需要额外判断：能被 4 整除但不能被 100 整除，或者能被 400 整除。如果是闰年且月份在 3 月及以后，结果需要加 1。

**代码实现：**
```go
func dayOfYear(date string) int {
	year, _ := strconv.Atoi(date[:4])
	month, _ := strconv.Atoi(date[5:7])
	day, _ := strconv.Atoi(date[8:10])
	count := [...]int{0, 31, 59, 90, 120, 151, 181, 212, 243, 273, 304, 334}
	// 判断闰年
	if ((year%4 == 0 && year%100 != 0) || year%400 == 0) && month >= 3 {
		return count[month-1] + day + 1
	}
	return count[month-1] + day
}
```

**复杂度分析：**
- 时间复杂度：O(1)，只涉及常数次字符串解析和计算
- 空间复杂度：O(1)，只使用了常数个额外变量
