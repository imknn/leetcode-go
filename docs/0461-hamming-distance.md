## 题目描述

两个整数之间的汉明距离指的是这两个数字对应二进制位不同的位置的数目。

给出两个整数 `x` 和 `y`，计算它们之间的汉明距离。

**注意：** 0 <= `x`, `y` < 2^31。

**示例：**
```
输入：x = 1, y = 4
输出：2
解释：
1   (0 0 0 1)
4   (0 1 0 0)
       ↑   ↑
上面的箭头指出了对应二进制位不同的位置。
```

**提示：**
- `0 <= x, y < 2^31`

## 解题算法

### 方法：异或 + 统计 1 的个数

**核心思路：**
1. 先对两个数进行异或运算，不同的位会变为 1，相同的位会变为 0
2. 然后统计异或结果中 1 的个数，即为汉明距离

**代码实现：**
```go
func hammingDistance(x int, y int) int {
	m := x ^ y // 异或计算不同的位置
	count := 0
	for m > 0 { // 统计1的个数
		if m&1 > 0 {
			count++
		}
		m >>= 1
	}
	return count
}
```

**复杂度分析：**
- 时间复杂度：O(1)，最多遍历 32 位
- 空间复杂度：O(1)，只使用常量额外空间
