## 题目描述

给你一个字符串 `s`，找到 `s` 中最长的 **回文子串**。

**示例 1：**
```
输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。
```

**示例 2：**
```
输入：s = "cbbd"
输出："bb"
```

**提示：**
- `1 <= s.length <= 1000`
- `s` 仅由数字和英文字母组成

## 解题算法

### 方法：中心扩展法

**核心思路：**
回文串一定关于中心对称，因此可以从每个可能的中心向两边扩展，找到以该点为中心的最长回文串。

**中心点有两种情况：**
1. **奇数长度回文**：以单个字符为中心，如 `"aba"` 的中心是 `'b'`
2. **偶数长度回文**：以相邻两字符间隙为中心，如 `"abba"` 的中心在两个 `'b'` 之间

**算法步骤：**
1. 遍历字符串的每个位置 `i`
2. 以 `i` 为中心扩展（奇数长度）
3. 以 `i` 和 `i+1` 为中心扩展（偶数长度）
4. 扩展时向两边移动，直到不是回文为止
5. 记录最长回文的起始位置和长度

**代码实现：**
```go
func longestPalindrome(s string) string {
    if len(s) < 2 {
        return s
    }
    start, maxLen := 0, 1

    expand := func(left, right int) {
        for left >= 0 && right < len(s) && s[left] == s[right] {
            if right-left+1 > maxLen {
                start = left
                maxLen = right - left + 1
            }
            left--
            right++
        }
    }

    for i := 0; i < len(s); i++ {
        expand(i, i)   // 奇数长度回文
        expand(i, i+1) // 偶数长度回文
    }

    return s[start : start+maxLen]
}
```

**复杂度分析：**
- 时间复杂度：O(n²)，其中 n 是字符串长度，需要遍历每个中心点并扩展
- 空间复杂度：O(1)，只使用常数额外空间

### 方法对比

| 方法 | 时间复杂度 | 空间复杂度 | 说明 |
|------|-----------|-----------|------|
| 暴力枚举 | O(n³) | O(1) | 枚举所有子串并判断回文 |
| 动态规划 | O(n²) | O(n²) | dp[i][j] 表示 s[i:j+1] 是否回文 |
| **中心扩展** | **O(n²)** | **O(1)** | **推荐，空间最优** |
| Manacher | O(n) | O(n) | 最优但实现复杂 |
