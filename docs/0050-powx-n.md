## 题目描述

实现 pow(x, n)，即计算 x 的 n 次幂函数（即 x^n）。

**示例 1：**
```
输入：x = 2.00000, n = 10
输出：1024.00000
```

**示例 2：**
```
输入：x = 2.10000, n = 3
输出：9.26100
```

**示例 3：**
```
输入：x = 2.00000, n = -2
输出：0.25000
解释：2^(-2) = 1/(2^2) = 1/4 = 0.25
```

## 解题算法

### 方法：快速幂（二进制拆分）

**核心思路：**
利用快速幂算法，将指数 n 进行二进制分解。每次将指数右移一位，如果当前位为 1 则累乘当前的底数。底数在每轮循环中自乘，实现 O(log n) 的幂运算。当 n 为负数时，先计算正数幂再取倒数。

**代码实现：**
```go
func myPow(x float64, n int) float64 {
	var i, total = n, float64(1)
	for i != 0 {
		if i&1 == 1 {
			total *= x
		}
		x *= x
		i /= 2
	}
	if n > 0 {
		return total
	} else {
		return 1 / total
	}
}
```

**复杂度分析：**
- 时间复杂度：O(log n)，快速幂每次将指数减半
- 空间复杂度：O(1)，只使用了常数个额外变量
