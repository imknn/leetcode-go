## 题目描述

给你一个 **非空** 整数数组 `nums`，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

你必须设计并实现线性时间复杂度的算法来解决此问题，且该算法只使用常量额外空间。

**示例 1：**
```
输入：nums = [2,2,1]
输出：1
```

**示例 2：**
```
输入：nums = [4,1,2,1,2]
输出：4
```

**示例 3：**
```
输入：nums = [1]
输出：1
```

**提示：**
- `1 <= nums.length <= 3 * 10^4`
- `-3 * 10^4 <= nums[i] <= 3 * 10^4`
- 除了出现次数大于 1 的那个元素以外，其余每个元素都只出现一次

## 解题算法

### 方法：异或运算

**核心思路：**
利用异或运算的性质：
1. **交换律**：`a ^ b ^ c <=> a ^ c ^ b`
2. **任何数与 0 异或为任何数**：`0 ^ n => n`
3. **相同的数异或为 0**：`n ^ n => 0`

将数组中所有元素进行异或运算，成对出现的元素会互相抵消变为 0，最终结果就是只出现一次的那个数字。

**代码实现：**
```go
func singleNumber(nums []int) int {
	n, length := nums[0], len(nums)
	for i := 1; i < length; i++ {
		n ^= nums[i]
	}
	return n
}
```

**复杂度分析：**
- 时间复杂度：O(n)，遍历数组一次
- 空间复杂度：O(1)，只使用常量额外空间
