## 题目描述

给定整数 n，返回所有小于非负整数 n 的质数的数量。

**示例 1：**
```
输入：n = 10
输出：4
解释：小于 10 的质数一共有 4 个，分别是 2, 3, 5, 7。
```

**示例 2：**
```
输入：n = 0
输出：0
```

**示例 3：**
```
输入：n = 1
输出：0
```

## 解题算法

### 方法：埃拉托斯特尼筛法

**核心思路：**
使用经典的埃氏筛法求质数。创建一个布尔数组标记每个数是否为合数，从 2 开始，将每个质数的倍数标记为合数。优化点：外层循环只需到 sqrt(n)，内层标记从 i*i 开始（因为更小的倍数已经被之前的质数标记过了）。

**代码实现：**
```go
func countPrimes(n int) int {
	nums := make([]bool, n)
	count := 0
	for i := 2; i*i < n; i++ {
		if !nums[i-1] {
			for j := i * i; j < n; j += i {
				nums[j-1] = true
			}
		}
	}
	for i := 2; i < n; i++ {
		if !nums[i-1] {
			count++
		}
	}
	return count
}
```

**复杂度分析：**
- 时间复杂度：O(n log log n)，埃氏筛法的经典时间复杂度
- 空间复杂度：O(n)，需要布尔数组存储每个数的标记
