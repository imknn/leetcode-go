## 题目描述

给你一个正整数 n，请你找出满足下述所有条件的最小整数：

- 该整数大于等于 n
- 该整数是 2 的倍数
- 该整数是 n 的倍数

如果存在这样的整数，返回它；否则，返回 -1。

**示例 1：**
```
输入：n = 5
输出：10
解释：大于等于 5 的最小的既是 2 的倍数又是 5 的倍数的数是 10。
```

**示例 2：**
```
输入：n = 6
输出：6
解释：6 是大于等于 6 的最小的同时是 2 和 3 的倍数的数。
```

**示例 3：**
```
输入：n = 10
输出：10
解释：10 是大于等于 10 的最小的同时是 2 和 5 的倍数的数。
```

## 解题算法

### 方法：奇偶判断

**核心思路：**
要同时是 2 的倍数和 n 的倍数，即求 2 和 n 的最小公倍数。当 n 为奇数时，LCM(2, n) = 2n；当 n 为偶数时，LCM(2, n) = n。通过位运算 `n&1` 判断奇偶性，奇数时左移一位（乘 2），偶数直接返回。

**代码实现：**
```go
func smallestEvenMultiple(n int) int {
	if n&1 == 1 { // 奇数
		return n << 1
	}
	return n
}
```

**复杂度分析：**
- 时间复杂度：O(1)，只涉及常数次位运算
- 空间复杂度：O(1)，只使用了常数个额外变量
