## 题目描述

给定一个整数 num，将其转化为 7 进制，并以字符串形式返回。

**示例 1：**
```
输入：num = 100
输出："202"
```

**示例 2：**
```
输入：num = -7
输出："-10"
```

**提示：**
- -10^7 <= num <= 10^7

## 解题算法

### 方法：进制转换

**核心思路：**
使用标准的进制转换方法：对 7 取模得到当前位的数字，对 7 整除得到剩余部分，重复直到剩余部分为 0。由于取模是从低位到高位，所以需要将结果反转。对于负数，先取绝对值处理，最后在结果前添加负号。

**代码实现：**
```go
func convertToBase7(num int) string {
	if num == 0 {
		return "0"
	}
	isNe := false
	if num < 0 {
		num = -num
		isNe = true
	}
	var buffer bytes.Buffer
	for num > 0 {
		buffer.WriteByte(byte('0' + (num % 7)))
		num /= 7
	}
	if isNe {
		buffer.WriteByte('-')
	}
	// 逆转
	s := buffer.Bytes()
	length := len(s)
	half := length / 2
	for i := 0; i < half; i++ {
		s[i], s[length-1-i] = s[length-1-i], s[i]
	}
	return buffer.String()
}
```

**复杂度分析：**
- 时间复杂度：O(log₇(n))，循环次数与 7 进制位数成正比
- 空间复杂度：O(log₇(n))，用于存储结果字符串
