## 题目描述

探险家小扣来到万灵之树前，拥有 `n` 颗幻境宝石 `gem[i]`。需要用链接节点和宝石组装成一棵二叉树：

- 链接节点作为非叶子节点，每个必须有 2 个子节点
- 宝石作为叶子节点，所有宝石都必须被使用

能量从根节点出发，按前序遍历规则移动并记录数字：
- 首次到达节点时记录 `1`
- 若为叶子节点，额外记录该宝石的数值
- 退出节点时记录 `9`

最终记录的数字依序连接成 `num`，满足 `num mod p = target` 即解开谜题。求有多少种组装方案。

**示例 1：**
```
输入：gem = [2,3], p = 100000007, target = 11391299
输出：1
```

**示例 2：**
```
输入：gem = [3,21,3], p = 7, target = 5
输出：4
```

**提示：**
- `1 <= gem.length <= 9`
- `0 <= gem[i] <= 10^9`
- `1 <= p <= 10^9`，保证 p 为素数
- `0 <= target < p`

## 解题算法

### 方法：块拼接 + DFS 枚举排列 + meet-in-the-middle

**核心思路：**

将满二叉树的前序遍历结果拆分为 `n+1` 个"块"（B0..Bn），每块的长度和值（mod p）由树形唯一确定。对每个树形，分别枚举左右两侧宝石排列，用掩码分桶 + 互补掩码配对实现 meet-in-the-middle 计数。

**算法步骤：**

1. **预处理**：计算每颗宝石的位数 `d[i]`、模值 `xi[i]`，以及 10 的幂 `pow10`

2. **块生成**（`buildBlocks`）：
   - 递归枚举所有 `Catalan(n)` 种满二叉树形状
   - 每种形状分解为 B0..Bn 块，记录每块的长度和值（mod p）
   - B0 = `'1'` + 左子树前缀，Bn = 右子树后缀 + `'9'`，中间块为叶子宝石贡献

3. **DFS 枚举排列**（`dfsEnum`）：
   - 对每棵树形，分别枚举左侧（`n/2` 个宝石）和右侧（`n-n/2` 个宝石）的所有排列
   - 沿途维护当前块值 `w`，按掩码分桶存储

4. **meet-in-the-middle 计数**：
   - 压缩每个桶为有序 `(key, count)` 数组
   - 枚举右侧掩码，用互补左侧掩码配对
   - 通过公式 `need = (target - base - aLeft * M_right) mod p` 计算所需值，二分查找计数

**块拼接示意（n=3, k=1）：**
```
树形:       [1]
           /   \
         [gem0] [1]
                /   \
            [gem1] [gem2]

块: B0='1', B1='9gem09gem19gem29', B2='9'
全数字: 1gem09gem19gem299
```

**复杂度分析：**
- 时间复杂度：O(Catalan(n) · 2^n · n)
- 空间复杂度：O(2^n)
