## 题目描述

实现 `myAtoi(string s)` 函数，将字符串转换为一个 32 位有符号整数（类似 C/C++ 中的 `atoi` 函数）。

读入字符串并丢弃无用的前导空格，检查下一个字符是否为符号位，如果是则记录符号，再读入连续的数字字符，将其转换为整数。如果超过 32 位有符号整数范围 `[−2^31, 2^31 − 1]`，需要截断。

**示例 1：**
```
输入：s = "42"
输出：42
```

**示例 2：**
```
输入：s = "   -42"
输出：-42
解释：第一个非空白字符为 '-'，它是一个负号。
```

**示例 3：**
```
输入：s = "4193 with words"
输出：4193
解释：截止在数字 '3' 之后读取字符 'w'，因为后面不是数字，所以停止。
```

**提示：**
- `0 <= s.length <= 200`
- `s` 由英文字母（大写和小写）、数字（0-9）、空格、`'+'`、`'-'` 和 `'.'` 组成

## 解题算法

### 方法：模拟

**核心思路：**
按照题目要求逐步处理字符串：
1. 跳过所有前导空格
2. 检查并记录正负符号
3. 逐个读取数字字符，每次将结果乘以 10 加上新数字
4. 在累加前检查是否会发生整数溢出，若溢出则返回边界值 `MaxInt32` 或 `MinInt32`

**代码实现：**
```go
func myAtoi(s string) int {
	i := 0
	n := len(s)

	// 1. 跳过前导空格
	for i < n && s[i] == ' ' {
		i++
	}

	// 如果已经到达字符串末尾，返回 0
	if i == n {
		return 0
	}

	// 2. 处理符号
	sign := 1
	if s[i] == '+' {
		i++
	} else if s[i] == '-' {
		sign = -1
		i++
	}

	// 3. 读取数字并处理溢出
	result := 0
	for i < n && s[i] >= '0' && s[i] <= '9' {
		digit := int(s[i] - '0')

		// 检查溢出：result * 10 + digit > math.MaxInt32
		if result > (math.MaxInt32-digit)/10 {
			if sign == 1 {
				return math.MaxInt32
			}
			return math.MinInt32
		}

		result = result*10 + digit
		i++
	}

	return sign * result
}
```

**复杂度分析：**
- 时间复杂度：O(n)，其中 n 是字符串长度，最多遍历一次
- 空间复杂度：O(1)，只使用了常数个变量
