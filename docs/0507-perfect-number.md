## 题目描述

对于一个正整数，如果它和除了自身以外的所有正因子之和相等，则称其为「完美数」。

给定一个整数 n，如果它是完美数，返回 true；否则，返回 false。

**示例 1：**
```
输入：num = 28
输出：true
解释：28 = 1 + 2 + 4 + 7 + 14，28 的所有因子（除了 28 本身）之和为 1 + 2 + 4 + 7 + 14 = 28。
```

**示例 2：**
```
输入：num = 7
输出：false
解释：7 的因子只有 1 和 7，所以不是完美数。
```

**提示：**
- 1 <= num <= 10^8

## 解题算法

### 方法：因子枚举

**核心思路：**
从 2 开始枚举到 sqrt(num)，对于每个能整除 num 的因子 a，同时收集因子 a 和 num/a。初始和为 1（1 始终是因子）。最后判断因子之和是否等于原数。注意小于 2 的数不可能是完美数。

**代码实现：**
```go
func checkPerfectNumber(num int) bool {
	if num < 2 {
		return false
	}
	// 先找出所有的因子
	var sum = 1
	for a := 2; a*a < num; a++ {
		if num%a == 0 {
			sum = sum + a + num/a
		}
	}
	return sum == num
}
```

**复杂度分析：**
- 时间复杂度：O(sqrt(n))，只需要枚举到 sqrt(n)
- 空间复杂度：O(1)，只使用了常数个额外变量
