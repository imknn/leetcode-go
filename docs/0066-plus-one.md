## 题目描述

给定一个由 **整数** 组成的 **非空** 数组所表示的非负整数，在该数的基础上加一。

最高位数字存放在数组的首位。 数组中每个元素只存储 **单个** 数字。

你可以假设除了整数 0 之外，这个整数不会以零开头。

**示例 1：**

```
输入：digits = [1,2,3]
输出：[1,2,4]
解释：输入数组表示数字 123。
```

**示例 2：**

```
输入：digits = [4,3,2,1]
输出：[4,3,2,2]
解释：输入数组表示数字 4321。
```

**示例 3：**

```
输入：digits = [0]
输出：[1]
```

## 解题算法

### 方法：模拟加法

**核心思路：**
从数组末尾开始遍历，模拟加一操作。如果当前位是 9，则将其置为 0 并产生进位；否则直接加 1 并退出循环。如果遍历完所有位仍有进位（即原数全是 9），则在数组头部插入 1。

**代码实现：**
```go
func plusOne(digits []int) []int {
	size, n := len(digits), 1
	for i := size - 1; i >= 0; i-- {
		if digits[i] == 9 {
			digits[i] = 0
			n = 1
		} else {
			digits[i] += n
			n = 0
			break
		}
	}
	if n == 1 {
		return append([]int{1}, digits...)
	}
	return digits
}
```

**复杂度分析：**
- 时间复杂度：O(n)
- 空间复杂度：O(1)（最坏情况需要 O(n) 的空间来创建新数组）
