## 题目描述

给你一个整数数组 `nums`，恰好有两个元素只出现一次，其余所有元素均出现两次。找出那只出现一次的两个元素。你可以按 **任意顺序** 返回答案。

你必须设计并实现线性时间复杂度的算法且仅使用常量额外空间来解决此问题。

**示例 1：**
```
输入：nums = [1,2,1,3,2,5]
输出：[3,5]
解释：[5, 3] 也是有效的答案。
```

**示例 2：**
```
输入：nums = [-1,0]
输出：[-1,0]
```

**示例 3：**
```
输入：nums = [1,2]
输出：[1,2]
```

**提示：**
- `2 <= nums.length <= 3 * 10^4`
- `-2^31 <= nums[i] <= 2^31 - 1`
- 除两个只出现一次的整数外，`nums` 中的其他数字都出现两次

## 解题算法

### 方法：哈希表统计

**核心思路：**
使用哈希表统计每个数字出现的次数，遍历哈希表找到出现次数为 1 的两个元素。

**代码实现：**
```go
func singleNumber03(nums []int) []int {
	result := make([]int, 2)
	m := make(map[int]int)
	for _, num := range nums {
		if _, ok := m[num]; !ok {
			m[num] = 1
		} else {
			m[num]++
		}
	}
	i := 0
	for k, v := range m {
		if v == 1 {
			result[i] = k
			i++
		}
		if i == 2 {
			break
		}
	}
	return result
}
```

**复杂度分析：**
- 时间复杂度：O(n)，遍历数组和哈希表各一次
- 空间复杂度：O(n)，哈希表存储元素及出现次数
