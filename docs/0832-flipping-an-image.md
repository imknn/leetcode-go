## 题目描述

给定一个 `n x n` 的二进制矩阵 `image`，先水平翻转图像，然后反转图像并返回结果。

水平翻转就是将每一行的顺序反转。反转是指将每个元素中的 0 变成 1，1 变成 0。

**示例 1：**

```
输入：image = [[1,1,0],[1,0,1],[0,0,0]]
输出：[[1,0,0],[0,1,0],[1,1,1]]
解释：首先水平翻转：[[0,1,1],[1,0,1],[0,0,0]]
然后反转：[[1,0,0],[0,1,0],[1,1,1]]
```

**示例 2：**

```
输入：image = [[1,1,0,0],[1,0,0,1],[0,1,1,1],[1,0,1,0]]
输出：[[1,1,0,0],[0,1,1,0],[0,0,0,1],[1,0,1,0]]
```

## 解题算法

### 方法：模拟（双指针同时翻转和反转）

**核心思路：**
对每一行使用双指针从两端向中间遍历。每次交换对称位置的元素（实现水平翻转），同时对两个元素进行取反（实现反转）。当两个指针相遇时，单独处理中间元素的反转。

**代码实现：**
```go
func flipAndInvertImage(A [][]int) [][]int {
	for _, m := range A {
		length := len(m)
		for i := 0; ; i++ {
			j := length - i - 1
			if i == j {
				if m[i] == 1 {
					m[i] = 0
				} else {
					m[i] = 1
				}
				break
			}
			if i > j {
				break
			}
			// 水平翻转
			m[i], m[j] = m[j], m[i]
			// 反转
			if m[i] == 1 {
				m[i] = 0
			} else {
				m[i] = 1
			}
			if m[j] == 1 {
				m[j] = 0
			} else {
				m[j] = 1
			}
		}
	}
	return A
}
```

**复杂度分析：**
- 时间复杂度：O(n^2)
- 空间复杂度：O(1)
