## 题目描述

给定一个整数 `n`，写一个函数来判断它是否是 4 的幂次方。如果是，返回 `true`；否则，返回 `false`。

**示例 1：**
```
输入：n = 16
输出：true
```

**示例 2：**
```
输入：n = 5
输出：false
```

**示例 3：**
```
输入：n = 1
输出：true
```

**提示：**
- `-2^31 <= n <= 2^31 - 1`

## 解题算法

### 方法：位运算

**核心思路：**
4 的幂和 2 的幂一样，二进制表示中只会出现一位 1。但 4 的幂的 1 总是出现在奇数位（从第 0 位开始计数）。

1. 首先判断是否是 2 的幂：`n > 0 && (n & (n-1) == 0)`
2. 然后用掩码 `0x55555555`（二进制 `01010101...`）检查 1 是否在奇数位上

**代码实现：**
```go
func isPowerOfFour(num int) bool {
	if num > 0 && (num&(num-1) == 0) {
		return num&0x55555555 > 0
	}
	return false
}
```

**复杂度分析：**
- 时间复杂度：O(1)，只进行常数次位运算
- 空间复杂度：O(1)，只使用常量额外空间
