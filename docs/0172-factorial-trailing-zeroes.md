## 题目描述

给定一个整数 n，返回 n! 结果中尾随零的个数。

**示例 1：**
```
输入：n = 3
输出：0
解释：3! = 6，尾随零为 0
```

**示例 2：**
```
输入：n = 5
输出：1
解释：5! = 120，尾随零为 1
```

**示例 3：**
```
输入：n = 0
输出：0
```

## 解题算法

### 方法：统计因子5的个数

**核心思路：**
尾随零由因子 2 和因子 5 配对产生。在 n! 中，因子 2 的个数一定多于因子 5 的个数，因此尾随零的个数等于因子 5 的个数。通过不断除以 5 来统计 n! 中因子 5 的总数（包括 5、25、125 等多次因子 5 的贡献）。

**代码实现：**
```go
func trailingZeroes(n int) int {
	count := 0
	for n >= 5 {
		n /= 5
		count += n
	}
	return count
}
```

**复杂度分析：**
- 时间复杂度：O(log₅(n))，每次将 n 除以 5
- 空间复杂度：O(1)，只使用了常数个额外变量
