## 题目描述

罗马数字包含七种基本字符：`I(1)`, `V(5)`, `X(10)`, `L(50)`, `C(100)`, `D(500)`, `M(1000)`。

通常情况下，小数字在大数字的右边表示相加（如 `II = 2`）。但也有减法规则：`I` 放在 `V` 或 `X` 前面表示 4 或 9，`X` 放在 `L` 或 `C` 前面表示 40 或 90，`C` 放在 `D` 或 `M` 前面表示 400 或 900。

给定一个罗马数字字符串，将其转换为整数。

**示例 1：**
```
输入：s = "III"
输出：3
```

**示例 2：**
```
输入：s = "LVIII"
输出：58
```

**示例 3：**
```
输入：s = "MCMXCIV"
输出：1994
```

**提示：**
- `1 <= s.length <= 15`
- `s` 仅包含字符 `('I', 'V', 'X', 'L', 'C', 'D', 'M')`
- 题目数据保证 `s` 是一个有效的罗马数字，且表示范围在 `[1, 3999]`

## 解题算法

### 方法：前缀匹配（贪心）

**核心思路：**
将所有可能的罗马数字值（包括减法组合如 CM、CD 等）从大到小排列。从头开始匹配字符串，如果字符串以某个罗马数字前缀开头，就将其对应值加到结果中，并从字符串中去掉该前缀。

**代码实现：**
```go
func romanToInt(s string) int {
	number := []int{1000, 900, 500, 400, 100, 90, 50, 40, 10, 9, 5, 4, 1}
	roman := []string{"M", "CM", "D", "CD", "C", "XC", "L", "XL", "X", "IX", "V", "IV", "I"}
	var num int
	for idx := 0; len(s) > 0 && idx < 13; {
		if strings.HasPrefix(s, roman[idx]) {
			num = num + number[idx]
			s = s[len(roman[idx]):]
		} else {
			idx++
		}
	}
	return num
}
```

**复杂度分析：**
- 时间复杂度：O(n)，其中 n 是字符串长度，每个字符最多被处理一次
- 空间复杂度：O(1)，只使用了常数个变量
