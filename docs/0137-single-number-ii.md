## 题目描述

给你一个整数数组 `nums`，除某个元素只出现一次以外，其余每个元素均出现三次。找出那个只出现了一次的元素。

你必须设计并实现线性时间复杂度的算法来解决此问题，且该算法只使用常量额外空间。

**示例 1：**
```
输入：nums = [2,2,3,2]
输出：3
```

**示例 2：**
```
输入：nums = [0,1,0,1,0,1,99]
输出：99
```

**提示：**
- `1 <= nums.length <= 3 * 10^4`
- `-2^31 <= nums[i] <= 2^31 - 1`
- 除某个元素只出现一次以外，其余每个元素均出现三次

## 解题算法

### 方法：哈希表统计

**核心思路：**
使用哈希表统计每个数字出现的次数，遍历哈希表找到出现次数为 1 的元素即可。

**其他方法（注释中有提及）：**
1. 将数字转成 32 位二进制，统计各个位上 1 的数量，不是 3 的倍数的位写 1，其他位写 0
2. 使用两个变量 `n1`、`n0` 统计每一位的数量，模拟状态机转换

**代码实现：**
```go
func singleNumber3(nums []int) int {
	m, n := make(map[int]int), nums[0]
	for _, num := range nums {
		if _, ok := m[num]; !ok {
			m[num] = 1
		} else {
			m[num]++
		}
	}
	for k, v := range m {
		if v == 1 {
			return k
		}
	}
	return n
}
```

**复杂度分析：**
- 时间复杂度：O(n)，遍历数组和哈希表各一次
- 空间复杂度：O(n)，哈希表存储元素及出现次数
