## 题目描述

有效括号字符串为空 `""`、`"(" + A + ")"` 或 `A + B`，其中 `A` 和 `B` 都是有效的括号字符串，`+` 代表字符串的连接。

例如，`""`，`"()"`，`"(())()"` 和 `"(()(()))"` 都是有效的括号字符串。

如果有效字符串 `S` 非空，且没有将其拆分为 `A + B` 的方法，我们称其为原语（primitive），其中 `A` 和 `B` 都是非空有效括号字符串。

给出一个非空有效字符串 `S`，考虑将其进行原语分解，即：将 `S` 分解为若干个原语（`S = P_1 + P_2 + ... + P_k`），并且将其最外层的括号删掉。

返回删除最外层括号后的结果字符串。

**示例 1：**

```
输入：s = "(()())(())"
输出："()()()"
解释：输入字符串为 "(()())(())"，原语分解为 "(()())" + "(())"。删除每个部分的最外层括号后得到 "()()" + "()" = "()()()"。
```

**示例 2：**

```
输入：s = "(()())(())(()(()))"
输出："()()()()(())"
```

**示例 3：**

```
输入：s = "()()"
输出：""
```

## 解题算法

### 方法：计数法

**核心思路：**
使用一个计数器 `size` 来跟踪当前的括号深度。遇到 `(` 时计数加一，遇到 `)` 时计数减一。当计数归零时，表示一个原语结束。在构建结果字符串时，跳过每个原语的第一个 `(` 和最后一个 `)`（即最外层括号）。通过记录每个原语的起始位置 `left`，可以高效地截取子串。

**代码实现：**
```go
func removeOuterParentheses(S string) string {
	var buffer bytes.Buffer
	left, size := 0, 0
	for idx, ch := range S {
		if ch == '(' {
			size++
		} else {
			size--
		}
		if size == 0 {
			// 原语 S[left : idx+1]), 去除前后括号就是S[left+1 : idx]
			buffer.WriteString(S[left+1 : idx])
			left = idx + 1
		}
	}
	return buffer.String()
}
```

**复杂度分析：**
- 时间复杂度：O(n)，其中 n 是字符串的长度
- 空间复杂度：O(n)，用于存储结果字符串
