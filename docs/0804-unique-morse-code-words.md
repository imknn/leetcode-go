## 题目描述

国际摩尔斯密码定义一种标准编码方式，将每个字母映射到一系列点和划。例如，字母 `a` 被映射为 `".-"`，字母 `b` 被映射为 `"-..."`，以此类推。

给定一个字符串数组 `words`，每个单词都可以写成每个字母对应摩尔斯码的组合。例如，`"cab"` 可以写成 `"-.-..--.."`。将所有字母的编码组合在一起。

返回不同单词的编码组合的数量。

**示例 1：**
```
输入：words = ["gin", "zen", "gig", "msg"]
输出：2
解释：每种组合可以表示为 "gig" 和 "msg" 的编码都是 "g..---...---."，所以 "gin" 和 "zen" 的编码都是 "g..---...---."。
但实际上 "gin" 的编码是 "--...---."，"zen" 的编码是 "--...---.."，它们不同。
```

**示例 2：**
```
输入：words = ["a"]
输出：1
```

**提示：**
- `1 <= words.length <= 100`
- `1 <= words[i].length <= 12`
- `words[i]` 仅由小写英文字母组成

## 解题算法

### 方法：哈希集合去重

**核心思路：**
使用摩尔斯码映射表将每个单词的每个字母转换为对应的摩尔斯码，拼接后存入哈希集合。最终集合的大小即为不同编码的数量。

**代码实现：**
```go
func uniqueMorseRepresentations(words []string) int {
	dir := [...]string{".-", "-...", "-.-.", "-..", ".", "..-.", "--.", "....", "..", ".---", "-.-", ".-..", "--", "-.", "---", ".--.", "--.-", ".-.", "...", "-", "..-", "...-", ".--", "-..-", "-.--", "--.."}
	m := make(map[string]bool)
	for _, w := range words {
		var buffer bytes.Buffer
		for _, c := range w {
			buffer.WriteString(dir[c-'a'])
		}
		m[buffer.String()] = true
	}
	return len(m)
}
```

**复杂度分析：**
- 时间复杂度：O(N * K)，其中 N 是单词数量，K 是每个单词的平均长度
- 空间复杂度：O(N * K)，哈希集合存储不同的编码组合
