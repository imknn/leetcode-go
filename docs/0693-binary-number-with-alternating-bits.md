## 题目描述

给定一个正整数，检查它的二进制表示是否总是 0、1 交替出现：换句话说，就是二进制表示中相邻两位的数字永不相同。

**示例 1：**
```
输入：n = 5
输出：true
解释：5 的二进制表示是：101
```

**示例 2：**
```
输入：n = 7
输出：false
解释：7 的二进制表示是：111
```

**示例 3：**
```
输入：n = 11
输出：false
解释：11 的二进制表示是：1011
```

**提示：**
- `1 <= n <= 2^31 - 1`

## 解题算法

### 方法：逐位检查

**核心思路：**
从最低位开始，每次右移一位，检查相邻两位是否相同。如果发现相邻两位相同，则不是交替位二进制数。

**代码实现：**
```go
func hasAlternatingBits(n int) bool {
	t := n % 2
	for n >>= 1; n > 0; n >>= 1 {
		k := n % 2
		if t == k {
			return false
		}
		t = k
	}
	return true
}
```

**复杂度分析：**
- 时间复杂度：O(1)，最多遍历 32 位
- 空间复杂度：O(1)，只使用常量额外空间
