## 题目描述

给定一个整数 n，判断它是否是 3 的幂次方。如果是，返回 true；否则，返回 false。

**示例 1：**
```
输入：n = 27
输出：true
```

**示例 2：**
```
输入：n = 0
输出：false
```

**示例 3：**
```
输入：n = 9
输出：true
```

**示例 4：**
```
输入：n = 45
输出：false
```

## 解题算法

### 方法：循环整除

**核心思路：**
如果 n 是 3 的幂次方，那么不断将 n 除以 3，最终结果应该为 1。如果 n 不能被 3 整除，则一定不是 3 的幂。注意需要先排除 n=0 的情况，因为 0 不是 3 的幂。

**代码实现：**
```go
func isPowerOfThree(n int) bool {
	if n == 0 {
		return false
	}
	for n%3 == 0 {
		n /= 3
	}
	return n == 1
}
```

**复杂度分析：**
- 时间复杂度：O(log₃(n))，每次将 n 除以 3
- 空间复杂度：O(1)，只使用了常数个额外变量
