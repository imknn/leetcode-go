## 题目描述

有一个单链表的头节点 `head`，我们想删除其中的一个节点 `node`。

给你一个需要删除的节点 `node`。你将 **无法访问** 第一个节点 `head`。

所有值都是 **唯一的**，并且保证给定的节点 `node` 不是链表中的最后一个节点。

注意，删除节点并不是指从内存中删除它。这里的意思是：

- 给定节点的值不应该存在于链表中。
- 链表中的节点数应该减少 1。
- `node` 前面的所有值保持不变。
- `node` 后面的所有值保持不变。

**示例 1：**

```
输入：head = [4,5,1,9], node = 5
输出：[4,1,9]
解释：指定值为 5 的第二个节点，调用函数后，链表变为 4 -> 1 -> 9
```

**示例 2：**

```
输入：head = [4,5,1,9], node = 1
输出：[4,5,9]
解释：指定值为 1 的第三个节点，调用函数后，链表变为 4 -> 5 -> 9
```

## 解题算法

### 方法：节点复制

**核心思路：**
由于无法访问前一个节点，无法通过常规方式删除。但可以将下一个节点的值复制到当前节点，然后删除下一个节点，从而达到"删除当前节点"的效果。本质上是用后继节点替换当前节点。

**代码实现：**
```go
func deleteNode(node *ListNode) {
	node.Val = node.Next.Val
	node.Next = node.Next.Next
}
```

**复杂度分析：**
- 时间复杂度：O(1)，只进行了常数次操作
- 空间复杂度：O(1)，没有使用额外空间
