## 题目描述

编写一个函数，其作用是将输入的字符串反转过来。输入字符串以字符数组 `s` 的形式给出。

你必须在原地修改输入数组，使用 O(1) 的额外空间。

**示例 1：**
```
输入：s = ["h","e","l","l","o"]
输出：["o","l","l","e","h"]
```

**示例 2：**
```
输入：s = ["H","a","n","n","a","h"]
输出：["h","a","n","n","a","H"]
```

**提示：**
- `1 <= s.length <= 10^5`
- `s[i]` 都是可打印的 ASCII 字符

## 解题算法

### 方法：双指针交换

**核心思路：**
使用左右两个指针分别指向数组的首尾，每次交换两个指针所指的元素，然后左指针右移、右指针左移，直到两指针相遇。

**代码实现：**
```go
func reverseString(s []byte) {
	length := len(s)
	half := length / 2
	for i := 0; i < half; i++ {
		s[i], s[length-1-i] = s[length-1-i], s[i]
	}
}
```

**复杂度分析：**
- 时间复杂度：O(n)，遍历半个数组
- 空间复杂度：O(1)，原地交换，不使用额外空间
