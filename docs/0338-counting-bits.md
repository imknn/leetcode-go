## 题目描述

给你一个整数 `n`，对于 `0 <= i <= n` 中的每个 `i`，计算其二进制表示中 **1 的个数**，返回一个长度为 `n + 1` 的数组 `ans` 作为答案。

**示例 1：**
```
输入：n = 2
输出：[0,1,1]
解释：
0 --> 0
1 --> 1
2 --> 10
```

**示例 2：**
```
输入：n = 5
输出：[0,1,1,2,1,2]
解释：
0 --> 0
1 --> 1
2 --> 10
3 --> 11
4 --> 100
5 --> 101
```

**提示：**
- `0 <= n <= 10^5`

## 解题算法

### 方法：动态规划 + 位运算

**核心思路：**
利用前一个数字的 1 的个数来推导当前数字。通过逐位检查前一个数字 `i-1` 的二进制表示，找到最低位的 0 进行翻转来得到 `i` 的 1 的个数。

具体来说，从 `i-1` 的最低位开始：
- 如果是 1，则 1 的个数减 1（这个 1 会被翻转为 0）
- 如果是 0，则 1 的个数加 1（这个 0 会被翻转为 1），然后停止

**代码实现：**
```go
func countBits(num int) []int {
	result, t := make([]int, num+1), 0
	// 判断上一个数字的最后一位
	// 如果是0，那就 + 1
	// 如果是0， 向前遍历每一位，找到1就-1，直到找到0 +1 结束
	for i := 1; i <= num; i++ {
		result[i] = result[i-1]
		for j := 0; j < 32; j++ {
			if t&1 == 1 {
				result[i]--
				t >>= 1
			} else {
				result[i]++
				break
			}
		}
		t = i
	}
	return result
}
```

**复杂度分析：**
- 时间复杂度：O(n * 32) = O(n)，对于每个数字最多检查 32 位
- 空间复杂度：O(n)，存储结果数组
