## 题目描述

给定一个 32 位有符号整数，将整数反转。

**示例 1：**
```
输入：x = 123
输出：321
```

**示例 2：**
```
输入：x = -123
输出：-321
```

**示例 3：**
```
输入：x = 120
输出：21
```

**注意：**
假设我们的环境只能存储得下 32 位有符号整数，其数值范围是 [−2^31, 2^31 − 1]。请根据这个假设，如果反转后整数溢出，那么就返回 0。

## 解题算法

### 方法：逐位反转

**核心思路：**
通过不断取整数的末位数字，构建反转后的数字。使用 int64 类型存储中间结果以避免溢出，并在每次操作后检查结果是否超出 32 位整数的范围。如果溢出则返回 0。

**代码实现：**
```go
func reverse(x int) int {
	// -2187483648 ~ 2187483647
	isNe, sum := int64(1), int64(0)
	if x < 0 {
		isNe = -1
		x = -x
	}
	for x > 0 {
		sum = sum*10 + int64(x%10)
		x /= 10
		if sum > math.MaxInt32 || (sum*isNe < math.MinInt32) {
			return 0
		}
	}
	return int(sum * isNe)
}
```

**复杂度分析：**
- 时间复杂度：O(log(x))，每次循环处理一位数字，循环次数与 x 的位数成正比
- 空间复杂度：O(1)，只使用了常数个额外变量
